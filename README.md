# 🧾 sistema-pdv

Sistema de gestão para pequenos comércios — controle de vendas, pagamentos e fiados.

O projeto nasceu de uma necessidade real: organizar a gestão de uma lanchonete que hoje depende de anotações em caderno. A proposta é substituir esse controle manual por um sistema simples e confiável, pensado para ser adaptável a outros ramos parecidos (padarias, pizzarias, açougues).

> 🚧 Projeto em desenvolvimento.

---

## 🎯 O problema

Pequenos comércios costumam controlar tudo no papel, o que gera:
- Falta de controle do fechamento financeiro (quanto entra e quanto sai)
- Vendas fiadas esquecidas — dinheiro que se perde
- Falta de visão sobre vendas e pagamentos

## ✅ Escopo da versão 1

- Cadastro de produtos
- Registro de vendas com seus itens
- Controle de pagamentos, incluindo fiado com pagamento parcial
- Consulta de vendas em aberto (clientes devendo)

## 🔭 Planejado para versões futuras

- Controle de estoque
- Organização de pedidos em tempo real
- Suporte a múltiplos ramos de comércio
